Menus

To wrap up our game, we’re going to create menus which make the game look much more like a finalized product. Instead of creating our own menu system, we’re going to leverage the UI technologies at our fingertips: HTML and CSS. By using HTML and CSS, we can easily create declarative UIs that allow for images, buttons, text, and smooth animations.
