Gaining more control

Divs

We know how to style some element like h1 or a, but what if we want to style a group of elements? When you want to define styles for a part of your web page, but not the entire page, you’ll want to use a div.

The div element doesn’t have any pre-defined styles or meaning, making it great for custom parts of our page. Let’s try using a div to set a background on some links.

Classes

Divs are a great addition to our website, as they allow us to specify styling for only a part of our page. But how useful are divs if we can only specify one style for all of the divs on our page? Not very. Lucky for us, we can solve this problem by using classes.

Exercise #4 - http://pastebin.com/8sMKCA9p

Sometimes the best way to learn is by example. In this example, we can see that class is an attribute of the div element. Classes don’t only work on divs though, we can set them on any element inside of the HTML body. The other thing to note is there are no predefined class names in HTML, therefore we can use whatever class names we would like. We can see how we add a class to an element, so now let’s head back to our CSS to style the class.

Looking at our CSS now, we can see that in order to target a class instead of an element type, we use a single dot before the class name.

Margin / Padding

One important CSS concept to understand when spacing elements is the difference between margin and padding. Margin is the distance the element will be spaced from other elements. Padding is the distance inside of the element from the content to the end of the element. This is easiest to see when the element has a colored background. Here’s a couple illustrations to hopefully clarify this concept. This will be demonstrated in the next exercise.

Exercise #6 - http://pastebin.com/Y92642Zg

If you look back at your web page now, you’ll see that the first div, grey-bg, is now spaced 20 pixels from the left, 20 pixels from the right, and 20 pixels from the top and bottom. All of this space pushes it away from other elements. Meanwhile, the second div with the class brown-bg now has an extra 20 pixels of space on each side within the element. Try changing the values or combine margin and padding in one class to fully understand the difference.

Positioning

Our divs are looking a little better with different backgrounds, but there’s a lot of wasted space. How would we go about positioning them side by side to reclaim some of the unused space?

We aren’t the first people to want to align the divs on a website, there are many already established libraries for achieving this task. The most popular being Bootstrap, which is what we’re going to use.

Bootstrap

The sizing in Bootstrap is a little different than our 49% width example. Instead of specifying a percentage, we specify how many columns we want to allot the element out of twelve.
